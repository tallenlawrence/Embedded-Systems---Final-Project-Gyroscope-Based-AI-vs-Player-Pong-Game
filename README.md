# Embedded-Systems---Final-Project-Gyroscope-Based-AI-vs-Player-Pong-Game
Our motivation behind this project was to take an old arcade game, in this case pong, and add a more advanced modern touch to it. Here the idea was to create a controller that utilised level detecting to move the pong paddle, we also aimed to have an ai opponent with selectable difficulty to ensure the user can try new challenges. We also wanted the controller to be as interactive as possible, so we achieved this by adding a PBS and RPG to allow even more user input and for the system to be fully controllable from just the controller alone

Components used: Arduino Uno (ATMega328P), MPU 6050 Gyroscope, 1602 LCD, RPG
